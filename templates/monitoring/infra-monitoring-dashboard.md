# Infra Monitoring Dashboard Template

## Sections

1. Global health summary
2. Region/cloud filters
3. Server status cards
4. Queue/latency metrics
5. Incident table
6. Logs panel
7. Alert timeline
8. Export/report action

## Prompt

Generate an infra monitoring dashboard for servers, queues, DNS, proxies, or IP pools. Include health cards, filters, incident table, logs panel, alert timeline, severity badges, and responsive layout.

# Privacy Policy

This repository collects and organizes public frontend template metadata, design guidance, and prompt recipes.

## Collected data

The collector may store public repository metadata, public URLs, license names, component names, categories, tags, design notes, and prompt patterns.

## Not collected

The collector must not collect private project files, private repositories, secrets, API keys, credentials, environment files, proprietary code, paid source code, or gated source code.

## License rule

Copy source code only when the license is clearly compatible: MIT, Apache-2.0, BSD-2-Clause, BSD-3-Clause, ISC, or CC0-1.0.

For unclear, paid, gated, or proprietary sources, store metadata and prompt notes only.

## Automation

Daily automation may update this repository directly on `main` for testing. Review commits before using generated templates in production.

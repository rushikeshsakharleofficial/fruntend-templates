# Framework Adapters

Use these rules when converting templates between targets.

## Targets

- React / TypeScript
- HTML / CSS / JavaScript
- PHP / Blade-style templates
- Vue
- Svelte
- Angular
- Astro

## Rules

- Preserve the selected design-md tokens.
- Preserve responsive behavior.
- Preserve accessibility.
- Avoid unnecessary dependencies.
- Include realistic usage examples.
- Keep components reusable.

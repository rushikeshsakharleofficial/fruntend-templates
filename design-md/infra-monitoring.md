# Infra Monitoring Design Profile

## Best use cases
Server monitoring, queue monitoring, proxy/IP status, DNS migration, uptime dashboards, cloud inventory.

## Color tokens
- Primary: blue
- Healthy: green
- Warning: amber
- Critical: red
- Background: white / dark graphite
- Border: neutral gray

## Typography
Readable compact sans-serif with monospace for server names, IPs, DNS records, queue names, and log output.

## Buttons
Operational actions, compact filter buttons, export/report controls.

## Cards
KPI health cards, service tiles, status rows, metric deltas, incident cards.

## Layout
Filters at top, metrics grid, resource table, log panel, incident timeline, export action.

## Prompt pattern
Generate an infra monitoring dashboard with server health cards, filters, incident table, logs panel, severity badges, and responsive layout.

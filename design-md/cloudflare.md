# Cloudflare Design Profile

## Best use cases
Infrastructure, DNS, network, security, edge, WAF, proxy, and monitoring dashboards.

## Color tokens
- Background: white or graphite dark
- Accent: orange / amber
- Secondary accent: blue for links and informational state
- Success: green
- Warning: amber
- Danger: red

## Typography
Enterprise-style sans-serif with dense but readable dashboard typography. Use monospace for DNS records, IPs, logs, and rule expressions.

## Buttons
Orange primary action, neutral outline secondary, destructive red for risky operations.

## Cards
Metric cards with health badges, graph strips, service status, and severity markers.

## Layout
Zone/project switcher, left navigation, resource table, status-first hierarchy, incident panels.

## Prompt pattern
Generate a Cloudflare-style infrastructure dashboard with orange accents, zone switcher, health cards, DNS/network tables, status badges, and responsive layout.

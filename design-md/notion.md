# Notion Design Profile

## Best use cases
Docs, workspace apps, CMS, notes, knowledge bases, internal tools, clean admin pages.

## Color tokens
- Background: warm white / soft dark
- Foreground: neutral black
- Accent: subtle gray or pastel highlight
- Border: very light neutral
- Muted: soft gray

## Typography
Document-friendly system sans-serif. Comfortable line height. Clear heading hierarchy.

## Buttons
Subtle ghost buttons, soft borders, low visual noise, clear hover state.

## Cards
Paper-like sections, soft borders, minimal shadows, content-first hierarchy.

## Layout
Document sidebar, nested navigation, clean editor/content areas, breadcrumbs.

## Prompt pattern
Generate a Notion-style workspace interface with calm neutral palette, document sidebar, clean content sections, subtle buttons, and responsive layout.

# Free Mantis and TailwindAdmin Design Profiles

Purpose: reusable design guidance for newly verified free MIT dashboard templates. Use these profiles as generation guidance only; do not copy paid/pro assets or gated blocks.

## Free Mantis Material Admin

- Source: https://github.com/codedthemes/mantis-free-react-admin-template
- License: MIT; free repository only.
- Color tokens: Material-inspired primary/secondary/surface/error variables, neutral dashboard backgrounds, clear success/warning/danger status colors.
- Typography: practical Material dashboard hierarchy, readable table/form labels, strong card titles and KPI values.
- Buttons: MUI-style contained, outlined, text, icon, and destructive variants with visible keyboard focus.
- Cards: Material admin cards, analytics panels, form cards, table cards, prompt-library/documentation cards.
- Layout: React/Next dashboard shell with sidebar, header, dashboard content grid, simple routed pages, and responsive mobile behavior.
- Spacing: Material-compatible 8px rhythm; compact enough for enterprise dashboards without crowding forms.
- Radius/shadow: Material elevation tokens, soft radius, border clarity in dense admin views.
- Dark/light mode notes: keep token-ready surfaces; do not assume Pro-only dark/light options from upstream comparison tables.
- Component usage rules: use free Mantis structure for dashboards, form-heavy admin pages, tables, analytics cards, simple user/profile/account pages, and prompt-library style docs.
- Prompt pattern: "Generate a free Mantis-style Material React admin dashboard with sidebar/header shell, KPI cards, MUI forms, responsive tables, analytics panels, documentation cards, and accessible Material spacing. Use only free MIT patterns and ignore Pro-only apps, auth providers, plugins, and design files."

## Free TailwindAdmin Multi-Framework Dashboard

- Source: https://github.com/Tailwind-Admin/free-tailwind-admin-dashboard-template
- License: MIT; free repository only.
- Color tokens: Tailwind neutral palette, CSS variables for shadcn-style theming, primary blue/indigo accent, semantic green/amber/red states.
- Typography: modern admin labels, clear dashboard headings, compact table and chart text, readable form labels.
- Buttons: shadcn-style primary/secondary/outline/ghost/destructive buttons, compact table actions, accessible focus rings.
- Cards: KPI widgets, chart panels, profile cards, auth cards, error-page cards, table containers.
- Layout: reusable sidebar + topbar + routed content shell across Next.js, React, Vue, Angular, and TanStack Start variants.
- Spacing: balanced Tailwind density for SaaS dashboards, CRM admin panels, ecommerce back offices, and internal tools.
- Radius/shadow: rounded-lg cards, subtle shadows, border-first dark-mode panels.
- Dark/light mode notes: support token-based dark mode; keep chart/status colors accessible in both modes.
- Component usage rules: use free framework-specific package folders as references for dashboard/profile/table/chart/auth/error pages; do not import paid services, gated files, or non-free assets.
- Prompt pattern: "Generate a free TailwindAdmin-style shadcn dashboard with sidebar/topbar shell, KPI cards, chart panels, profile page, tables, login/register screens, error pages, dark mode, and framework-specific clean Tailwind components. Use only free MIT repository patterns."

# GitHub Design Profile

## Best use cases
Developer tools, repositories, CI/CD dashboards, logs, activity timelines, code-review interfaces.

## Color tokens
- Background: white / dark neutral
- Accent: blue links and green create actions
- Border: neutral gray
- Success: green
- Warning: yellow/amber
- Danger: red

## Typography
System sans-serif with monospace for code, logs, commit IDs, branches, and file paths.

## Buttons
Green primary for create/confirm actions, gray secondary, icon buttons for dense controls.

## Cards
Repo-style bordered cards, activity cards, file tree panels, workflow status blocks.

## Layout
Tabs, file/tree views, activity timelines, issue tables, side panels.

## Prompt pattern
Generate a GitHub-style developer dashboard with repo-like cards, tabs, activity feed, workflow statuses, file/log views, and responsive layout.

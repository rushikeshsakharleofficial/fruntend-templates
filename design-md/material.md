# Material Design Profile

## Best use cases
Enterprise apps, admin panels, forms, mobile responsive web apps, settings heavy products.

## Colors
Use a primary brand color, neutral surfaces, clear outline borders, success, warning, and error states.

## Typography
Use readable sans-serif hierarchy, clear form labels, and consistent body sizes.

## Buttons
Include filled, tonal, outlined, text, and icon button variants with focus and disabled states.

## Cards
Use outlined or elevated cards with clear grouping and consistent spacing.

## Layout
Use app bar, navigation rail or sidebar, responsive grid, and clear form sections.

## Prompt pattern
Generate a Material-style interface with structured layout, accessible controls, filled and outlined buttons, form clarity, and responsive grid.

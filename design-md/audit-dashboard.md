# Audit Dashboard Design Profile

Use dark graphite surfaces, cyan or lime accent, red alert state, amber warning state, and green success state.
Use compact sans-serif typography with monospace for logs, IPs, event IDs, and command output.
Use high contrast primary buttons, compact filters, dense metric cards, severity badges, timeline rows, and log panels.

Prompt: Generate an audit dashboard with dark UI, severity badges, log panels, event timeline, compact metrics, and responsive layout.

# Vercel Design Profile

## Best use cases
Developer SaaS, deployment tools, hosting dashboards, AI products, product landing pages.

## Color tokens
- Background: white / near-black
- Foreground: black / white
- Accent: minimal blue only for focus or links
- Border: light neutral gray
- Success: green
- Warning: amber
- Danger: red

## Typography
Use a clean modern sans-serif with tight headings and readable body text. Use monospace for code, API keys, logs, and deployment data.

## Buttons
Primary buttons should be black in light mode and white in dark mode. Secondary buttons should be bordered. Ghost buttons should be minimal.

## Cards
Flat cards, thin borders, subtle shadow, strong whitespace, compact metadata.

## Layout
Centered marketing pages, dashboard sidebar for apps, strong whitespace, product screenshot sections, clear CTA hierarchy.

## Prompt pattern
Generate a Vercel-style interface with minimal neutral palette, sharp typography, strong whitespace, compact cards, responsive layout, and dark/light mode.

# Slack Design Profile

## Best use cases
Chat, collaboration, team dashboards, member management, notifications, support inboxes.

## Color tokens
- Primary: aubergine or deep purple
- Accent: green, blue, yellow highlights
- Surface: clean white / dark neutral
- Border: soft neutral

## Typography
Friendly product typography with readable message and member content.

## Buttons
Rounded primary buttons, friendly secondary controls, clear hover and focus states.

## Cards
Conversation blocks, member cards, channel panels, notification cards.

## Layout
Workspace sidebar, channel list, member panels, message/activity feeds.

## Prompt pattern
Generate a Slack-style collaboration interface with workspace sidebar, channel/member sections, friendly rounded controls, and responsive layout.

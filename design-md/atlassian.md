# Atlassian Design Profile

## Best use cases
Enterprise SaaS, team management, project administration, permissions, issue workflows, internal dashboards.

## Color tokens
- Primary: blue
- Surface: white / dark neutral
- Border: enterprise neutral
- Success: green
- Warning: amber
- Danger: red

## Typography
Readable product typography with clear labels, tables, and status text.

## Buttons
Blue primary, neutral secondary, red destructive, compact icon actions.

## Cards
Structured panels, status lozenges, settings groups, admin cards.

## Layout
Project sidebar, settings navigation, admin tables, workflow pages, permission panels.

## Prompt pattern
Generate an Atlassian-style enterprise SaaS dashboard with blue accents, structured panels, settings navigation, status labels, and admin workflows.

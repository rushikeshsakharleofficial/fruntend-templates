# Linear Design Profile

## Best use cases
Project management, issue tracking, roadmap dashboards, command-palette driven SaaS apps.

## Color tokens
- Background: charcoal / near-black for dark mode
- Accent: violet / indigo
- Border: soft dark neutral
- Muted: gray-blue
- Success: green
- Warning: amber
- Danger: red

## Typography
Modern compact sans-serif. Tight labels, strong item titles, readable issue descriptions.

## Buttons
Dark primary, violet focus ring, compact icon buttons, keyboard-first controls.

## Cards
Layered dark cards, slight gradients, subtle border glow, compact metadata.

## Layout
Command palette, issue list, split-pane detail view, keyboard-friendly navigation.

## Prompt pattern
Generate a Linear-style product dashboard with dark surfaces, violet accents, command palette, issue list, status labels, and responsive layout.

# Shopify Design Profile

## Best use cases
Ecommerce storefronts, merchant dashboards, inventory admin, orders, customers, product pages.

## Color tokens
- Primary: green or black
- Surface: clean white
- Border: neutral gray
- Success: green
- Warning: amber
- Danger: red

## Typography
Commerce-friendly sans-serif with strong product names, readable prices, and clear admin labels.

## Buttons
Green or black primary action, neutral secondary, compact admin actions, clear destructive states.

## Cards
Product cards, order cards, compact metric cards, inventory panels.

## Layout
Storefront sections plus admin views for products, orders, inventory, customers, discounts, and settings.

## Prompt pattern
Generate a Shopify-style ecommerce interface with product grid, cart/admin states, merchant dashboard patterns, inventory tables, and responsive layout.

# Supabase Design Profile

## Best use cases
Database apps, auth dashboards, backend platforms, developer consoles, project settings pages.

## Color tokens
- Background: dark-first slate/zinc
- Accent: emerald green
- Border: subtle gray-green neutral
- Success: emerald
- Warning: amber
- Danger: red

## Typography
Use compact sans-serif typography. Use monospace for SQL, API keys, logs, and environment variables.

## Buttons
Emerald primary button, neutral secondary button, compact radius, clear disabled/loading states.

## Cards
Dark panels, thin borders, status chips, compact density, code-friendly sections.

## Layout
Project sidebar, organization switcher, dense tables, query/log sections, clear settings pages.

## Prompt pattern
Generate a Supabase-style dashboard with dark-first UI, emerald accents, project sidebar, compact cards, database/auth patterns, and responsive layout.
